# tic-tac-toe
A multiplayer Tic-Tac-Toe game written in Go. It uses websocket for client and server communication.

### Installation
    go get -u github.com/fredyw/tic-tac-toe
    
### Usage:
Server:
```
./tic-tac-toe server [--port]
```

Client:
```
./tic-tac-toe client [--host] [--port]
```
